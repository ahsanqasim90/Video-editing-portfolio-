# Aziz Sons Engineering Works Website Documentation

## Project Overview

This document summarizes the completed website for **Aziz Sons Engineering Works**, a Lahore-based industrial engineering and manufacturing company.

The website has been designed as a premium B2B industrial company website, focused on presenting the company's engineering expertise, manufacturing capabilities, leadership, projects, contact details, and location in a professional and modern way.

## Live Website

Live URL:

https://video-editing-portfolio-eta.vercel.app

Note: The website is currently hosted on a temporary Vercel URL. A custom domain can be connected later when the domain is purchased.

## Website Type

The website is currently built as a **single-page responsive website**.

All navigation links scroll smoothly to different sections on the same page instead of opening separate pages.
